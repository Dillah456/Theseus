# Dimensi Penelitian
1. Tujuan 
2. Manfaat
3. Waktu
4. Teknik pengumpulan data

# Berdasar Tujuan 
1. Penelitian Eksploratif
   Mencoba menggali informasi atau permasalahan yang relatif masih baru
2. Penelitian Deskriptif
   Menggambarkan karakter suatu Variabel, Himpunan ( Kelompok ), atau Gejala Sosial, yang terjadi di Masyarakat.
3. Penelitian Eksplanatif
	Mengejelaskan mengapa suatu fenomena atau gejala sosial dapat terjadi dan bertujuan untuk menghubungkan pola - pola yang berbeda namun memiliki keterkaitan serta menghasilkan Kausalitas ( Nanang Martono (2016), Neuman(2003) )

# Berdasar Manfaat
1. Penelitian dasar
2. Penelitian Terapan

# Berdasarkan Waktu
1. Penelitian Longitudinal ( Antar Waktu )
	   1. Penelitian Kecenderungan ( Trend Research )
	   2. Panel ( Gejala sama dalam Waktu yang berbeda )
	   3. Kohort ( Kesamaan Karakter responden )
2. Cross Sectional

# Berdasarkan Tehnik Pengumpulan 
1. Penelitian Kuantitatif

	2. Eksperimen

	4. Analisis data Sekunder
