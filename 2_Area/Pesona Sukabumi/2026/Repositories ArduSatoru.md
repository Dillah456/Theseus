# Perihal ArduSatoru
Proyek ini adalah proyek berbasis Expert System untuk memberikan saran berdasarkan Jarak, Waktu, dan Biaya yang dipakai.
Sejauh ini memang full ChatGPT, dan tidak ada Dokumentasi UML, rencana akan disatukan dengan Oort sehingga .json nya akan meluas, begitupun meggunakan API lainnya
